# Visualiza-o_de_dados
Desafios de decisão para cada cenário abaixo, você receberá um conjunto de dados e uma pergunta de negócio. Analisar a pergunta e o tipo de dado.  Consultar o Data Viz Project ou From Data to Viz para identificar o tipo de gráfico mais adequado. Princípios de Gestalt, Tufte e/ou Cleveland-McGill.  Implementar o gráfico usando matplotlib e seaborn.
